# Problem Statement
Develop an application that allows users to chat with a culinary agent, enabling users to create accounts and store recipes. The app should integrate with an existing culinary agent that processes JSON responses, with placeholder sections included for future integration by another developer.

### [markdown] ARCHITECTURE.md > UI/UX
# UI/UX
The user interface will consist of a registration and login page, a dashboard where users can view and manage their recipes, and a chat interface for interacting with the culinary agent.

### [code] frontend/scripts.js > function sendMessage() {
function sendMessage() {
    const userInput = document.getElementById('user-input').value;
    const chatBox = document.getElementById('chat-box');
    const token = localStorage.getItem('token');
    chatBox.innerHTML += `<div>User: ${userInput}</div>`;

    fetch('http://127.0.0.1:5000/chat', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${token}`
        },
        body: JSON.stringify({ message: userInput })
    })
    .then(response => response.json())
    .then(data => {
        chatBox.innerHTML += `<div>Agent: ${data.message}</div>`;
    })
    .catch(error => console.error('Error:', error));
}
