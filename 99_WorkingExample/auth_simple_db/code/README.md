# Chat Recipe App

## Project Description
This web application allows users to interact with an AI to generate and store recipes. Users must register and log in to access the chat feature where interactions are centered around recipe generation. The application uses a simple authentication system to authorize users for accessing the chat functionality.

## Architecture Summary
The application follows a client-server architecture:
- **Frontend**: HTML, CSS, JavaScript for UI
- **Backend**: Python, Flask for HTTP request handling
- **Database**: SQLite for data persistence

## Tech Stack
- **Frontend**: HTML/CSS/JavaScript
- **Backend**: Python, Flask
- **Database**: SQLite

## How to Run

### Backend
1. Make sure Python and Flask are installed.
2. Navigate to the `backend/` directory.
3. Run `python app.py` to start the server.

### Frontend
1. Open `frontend/index.html` in a web browser.

### Usage
1. Register a new user account.
2. Log in with the registered credentials.
3. Start chatting with the AI to generate recipes.
