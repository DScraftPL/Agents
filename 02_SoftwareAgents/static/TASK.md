## Problem Statement
The task is to create a simple application that displays greetings to the user. The primary goal is to have an app that succinctly outputs the messages "Hello, World!" and "Goodbye, World!". This straightforward application aims to demonstrate basic output functionalities in software development.

## Objectives
- Develop an application that displays the message "Hello, World!" upon initiation.
- Ensure the application can also display the message "Goodbye, World!" when prompted.
